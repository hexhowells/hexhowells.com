Part 1

Each line consists of characters with some characters being integers, iterate over each character and check if it's a digit, this can be done in Python with the function string.isdigit() all the integers found can be appended to a string and the last and first characters in the string can be combined and added to the running total.

Part 2

Part 2 requires parsing not only the individual characters but substrings in order to find digits, this can be done in a hacky way by using Python's replace() function. Here we can replace each occurance of "one", "two", "three", etc with its respective digit then do the same as part 1.

However consider the edgecases: 


      twone
      oneeight
      sevenine

when replacing the substring with a digit we corrupt the other substring (e.g. "2ne" therefore to make our solution even more hacky we can keep the first and last character of the substring inserting the digit inbetween.


# convert the digit strings into single digit values
def convert(line):
  for i, digit in enumerate(['one', 'two', 'three', 'four', 'five', 'six', 'seven', 'eight', 'nine']):
    line = line.replace(digit, digit[0] + str(i+1) + digit[-1])

  return line
